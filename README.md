# Campus Carpool Platform

## 环境安装和使用
 
 * python3，依赖库安装pip install -r reqirement.txt
 * MySQL 创建 taxi 数据库，在 `app/config.py` 中配置连接参数，替代文件中的用户名和密码即可。
 * 运行入口在 `taxiforcollege.py`，运行之后会出现 http://localhost:8000/ ，点击即可访问网站。
